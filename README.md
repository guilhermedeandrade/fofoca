# Fofoca

`fofoca` is a chat app built with Node, TypeScript and Socket.IO.

## Setup

Install all the dependencies:

```sh
yarn

# npm i
```

Run the development server:

```sh
yarn dev

# npm run dev
```

## Usage

Now you're able to test the application at [localhost:3000](http://localhost:3000)
